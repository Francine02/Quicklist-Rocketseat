**Quicklist** é um desafio do curso de **JavaScript - Rocketseat**, nesse desafio é necessário desenvolver um site simples para gerenciar uma lista de compras da semana. O usuário poderá adicionar novos itens, marcar como concluídos e também remover da lista quando não forem mais necessários

## Funcionalidades:
- Adicionar novos itens na lista
- Marcar item como concluído
- Remover item da lista
- Armazenar tudo no **LocalStorage**
- Interface responsiva

---
## Tecnologias utilizadas:
- HTML, CSS, JavaScript e LocalStorage


---
## Como executar o projeto:
Clone o repositório:

```bash
git clone https://github.com/Francine02/Quicklist-Rocketseat.git
```

Abra o arquivo `index.html` ou arraste ele para o navegador.

Se preferir, confira aqui o projeto rodando:

```bash
https://francine02.github.io/Quicklist-Rocketseat/
```
